# Análise de dados com R

## Objetivo
Este repositório é dedicado às atividades relacionadas ao curso **Introdução à Análise de dados para pesquisa no SUS** ministrado pelo [Campus Virtual da Fiocruz](https://cursosqualificacao.campusvirtual.fiocruz.br/). 

## Ferramentas
1. [R](https://www.r-project.org/)
2. [RStudio](https://posit.co/download/rstudio-desktop)
3. [Tidyverse](https://tidyverse.org/)
4. [Ggplot2](https://r-graph-gallery.com/ggplot2-package.html)
